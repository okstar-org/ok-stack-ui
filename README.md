# 🎁 项目介绍

**OkStack是由OkStar(okstar.org)社区开发和维护的支持企业业务的通用管理平台，OkMSG相关所有数据操作统一由OkStack服务端（OkStack）管理，企业管理人员在OkMSG管理端中拥有对所有数据、业务和产品的完全管控能力,同时拥有更新的技术和性能。**
- 具备跨平台、拓展性。
- 可独立部署私有化系统。
- 兼容Windows、Linux等平台。
